# Teste Google Earth Engine

Esta seção documenta os testes e validações realizados com a API do Google Earth Engine.

## Objetivo

Validar a integração com o Google Earth Engine e testar a capacidade de obtenção e processamento de imagens de satélite para as métricas definidas.

## Atividades Realizadas

1. Configuração de credenciais e autenticação GEE
2. Testes de consulta a diferentes catálogos de imagens (Sentinel-2, Landsat)
3. Validação de cálculo de índices de vegetação
4. Análise de performance e otimização de queries
5. Tratamento de nuvens e máscaras de qualidade

## Resultados

Os testes demonstraram a viabilidade técnica da solução e permitiram ajustes na estratégia de processamento para otimizar tempo de resposta e qualidade dos dados.
