# Sprint 2 - Esboço Inicial

## Visão Geral

A segunda sprint focou na criação dos primeiros artefatos técnicos do projeto: modelagem do banco de dados, validação da integração com Google Earth Engine e desenvolvimento do protótipo inicial da interface.

## Entregas da Sprint

### 06 - Modelagem do Banco de Dados

Projeto e documentação da estrutura de dados que sustenta todas as funcionalidades da plataforma.

[Acessar documentação →](./06_Modelagem_Banco/README.md)

### 07 - Teste Google Earth Engine

Validação técnica da integração com a API do Google Earth Engine e testes de processamento de imagens de satélite.

[Acessar documentação →](./07_Teste_Google_Earth_Engine/README.md)

### 08 - Protótipo Frontend

Desenvolvimento do protótipo inicial da interface do usuário com as principais telas e fluxos de navegação.

[Acessar documentação →](./08_Prototipo_Frontend/README.md)

## Resultados Obtidos

Ao final desta sprint, a equipe tinha:

- ✅ Banco de dados modelado e documentado
- ✅ Integração com Google Earth Engine validada
- ✅ Protótipo funcional da interface
- ✅ Principais fluxos de usuário implementados
- ✅ Fundação técnica estabelecida para desenvolvimento
