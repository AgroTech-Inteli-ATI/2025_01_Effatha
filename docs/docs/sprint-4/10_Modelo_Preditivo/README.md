# Modelo Preditivo

Esta seção documenta o desenvolvimento e implementação do modelo preditivo para análise temporal de métricas agrícolas.

## Objetivos da Sprint 4

Durante esta sprint, focamos em:

1. Desenvolvimento de modelos de séries temporais (SARIMA)
2. Previsão de tendências de crescimento vegetal
3. Análise preditiva de métricas como NDVI, biomassa e cobertura vegetal
4. Criação de API para servir previsões
5. Integração com o frontend para visualização de dados preditivos
