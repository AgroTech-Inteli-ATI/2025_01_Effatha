# Sprint 4 - Modelo Preditivo

## Visão Geral

A quarta sprint introduziu capacidades preditivas ao sistema, permitindo não apenas análise histórica, mas também previsão de tendências futuras das métricas agrícolas.

## Entregas da Sprint

### 10 - Modelo Preditivo

Desenvolvimento e implementação de modelos de séries temporais para previsão de métricas agrícolas.

[Acessar documentação →](./10_Modelo_Preditivo/README.md)

## Resultados Obtidos

Ao final desta sprint, a equipe tinha:

- ✅ Modelos SARIMA implementados e validados
- ✅ API de previsões funcional
- ✅ Integração com frontend para visualização
- ✅ Previsões de NDVI, biomassa e cobertura vegetal
- ✅ Sistema completo com análise retrospectiva e preditiva
