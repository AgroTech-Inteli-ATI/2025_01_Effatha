# Variáveis Agrícolas

Esta seção documenta as variáveis e métricas agrícolas que o sistema é capaz de mensurar através de análise de imagens de satélite.

## Principais Métricas

O sistema trabalha com diversas métricas derivadas de sensoriamento remoto:

### Índices de Vegetação

- **NDVI** (Normalized Difference Vegetation Index)
- **EVI** (Enhanced Vegetation Index)
- **NDWI** (Normalized Difference Water Index)
- **NDMI** (Normalized Difference Moisture Index)
- **SAVI** (Soil Adjusted Vegetation Index)

### Métricas Complementares

- Biomassa vegetal
- Cobertura vegetal
- Saúde das plantas
- Estresse hídrico

## Fundamentação Científica

Cada variável foi selecionada com base em literatura científica consolidada e validação com especialistas da área agrícola.
