# Personas e Jornada do Usuário

Esta seção apresenta as personas definidas para o projeto e suas respectivas jornadas de uso do sistema.

## Objetivo

Compreender profundamente os diferentes perfis de usuários que utilizarão a plataforma, suas necessidades, dores e expectativas. Este entendimento guia o desenvolvimento de funcionalidades alinhadas com as necessidades reais dos usuários.

## Personas Identificadas

As personas representam os principais tipos de usuários da solução:

- Pesquisadores agrícolas
- Técnicos de campo
- Gestores de propriedades rurais
- Cientistas de dados especializados em agricultura
