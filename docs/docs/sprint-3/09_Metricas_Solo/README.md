# Métricas de Solo

Nesta seção, exploramos as métricas relacionadas às propriedades do solo que podem ser analisadas através de imagens de satélite e dados geoespaciais.

## Conteúdo

- [Teor de Argila](./teor-de-argila.md)

## Objetivos da Sprint 3

Durante esta sprint, focamos em:

1. Pesquisa e definição de métricas de solo relevantes para agricultura
2. Implementação de algoritmos para análise de propriedades do solo
3. Integração com Google Earth Engine para obtenção de dados
4. Validação dos resultados com dados reais
